# Roadmap

## Current State

Sharkie is a working MVP with a Rust API backend, PostgreSQL database, and a statically-exported Next.js frontend. The core functionality ‚Äî historical exchange rate tracking, charts, comparison, calculator ‚Äî is functional.

## Priority 1 ‚Äî Security Hardening

### Rate Limiting

Add `tower-governor` middleware to protect against abuse:

- History queries: 60 req/min per IP

### Security Headers

Add middleware for:

- `Strict-Transport-Security` (HSTS)
- `X-Content-Type-Options: nosniff`
- `X-Frame-Options: DENY`
- `Referrer-Policy: strict-origin-when-cross-origin`

### Body Size Limits

Add `tower-http::limit::RequestBodyLimitLayer` to prevent memory abuse from large request bodies.

### Request Timeouts

Add `tower::timeout::TimeoutLayer` for per-request deadlines (e.g., 30s).
