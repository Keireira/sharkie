# Frontend

The frontend is a Next.js 16 application with React 19, using the App Router with static export (`output: 'export'`). In production it's served as static files by Nginx.

## Stack

- **React 19** with React Compiler (automatic memoization)
- **styled-components** for CSS-in-JS with theming
- **TanStack React Query** for server state management
- **Framer Motion** for animations
- **Recharts** for charting
- **react-simple-maps** for the world map
- **i18next** for internationalization (4 languages)
- **Biome** for linting and formatting

## Theming

Two themes are available:

- **Dark** — near-black (#111111) with lime accent (#CAFF33)
- **Light** — warm white (#FFFAF5) with rainbow accents (red to blue)

Themes are managed through styled-components' `ThemeProvider` and toggled via the sidebar.

## API Integration

The backend serves an OpenAPI 3.1 spec at `/openapi.json`. This can be used to generate a typed API client for the frontend instead of hand-written fetch functions. Tools like `openapi-typescript` or `orval` can consume the spec to produce typed hooks compatible with React Query.

## State Management

### Settings (React Context)
User preferences stored in `localStorage`:
- Theme (dark/light)
- Language (en/ru/ja/es)
- Base currency
- Selected currencies
- Favorite currencies
- Period selection

### Server State (React Query)
All API data is managed through React Query with these query patterns:
- `['rates', from, to, base, currencies]` — main dashboard data
- `['rates-compare', mode, from, to, base, currencies]` — comparison period
- `['usd-rates', date]` — centralized USD-normalized cache

### Rate Store
The centralized rate store (`useRateStore.ts`) normalizes all fetched rates to USD base and caches them per-date. This allows the calculator and other components to compute arbitrary currency pairs without additional API calls.

## Code Conventions

Coding conventions:

- **Arrow functions** for all functions and components
- **No manual memoization** — React Compiler handles it (`useMemo`/`useCallback`/`React.memo` are prohibited)
- **`import type`** for type-only imports
- **Default exports** for components
- **styled-components** with `$transient` props and theme access
- **Biome** for linting and formatting (tabs, single quotes, 120 char width)
