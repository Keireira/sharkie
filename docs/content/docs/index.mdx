# Sharkie

Sharkie is a full-stack currency exchange rates dashboard that tracks 150+ fiat currencies and 12 cryptocurrencies with historical data going back to the year 2000.

## Key Features

- **Real-time rates** — fetched every 4 hours from an external currency API
- **Historical data** — daily exchange rates from January 1, 2000
- **Interactive charts** — area charts with normalization, brush navigation, and multi-currency comparison
- **Volatility heatmap** — visual grid showing daily percentage changes
- **Period comparison** — WoW, MoM, QoQ, HoH, YoY rate change analysis
- **Currency map** — interactive world map with currency overlays
- **Built-in calculator** — floating or full-screen converter with historical date support
- **Multi-language** — English, Russian, Japanese, Spanish
- **Dark/Light themes** — with distinct color palettes
- **PWA** — installable as a Progressive Web App
- **Static export** — no Node.js runtime required in production
