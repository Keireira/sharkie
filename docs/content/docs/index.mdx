# Sharkie

Sharkie is a full-stack currency exchange rates dashboard that tracks 150+ fiat currencies and 12 cryptocurrencies with historical data going back to the year 2000.

## Quick Start

```bash
# Start database
make dev-db

# Start backend (port 3000)
make dev-be

# Start frontend (port 3001)
make dev-fe

# Or start everything at once
make dev
```
